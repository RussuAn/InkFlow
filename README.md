# InkFlow — cучасна електронна бібліотека


**InkFlow** — це веб-платформа для читання, колекціонування та обміну книгами. Проєкт поєднує класичну любов до літератури з сучасними технологіями, дозволяючи користувачам створювати власні віртуальні полиці, читати PDF/EPUB файли онлайн та взаємодіяти з іншими читачами.

---

## Основні можливості

### Для читачів
- **Розумна читалка:** вбудований PDF-рідер, що автоматично запам'ятовує останню прочитану сторінку.
- **Особиста бібліотека:** сортування книг за статусами: "У планах", "Читаю", "Прочитано", "Покинуто".
- **Економіка та магазин:** внутрішня валюта (монети) для купівлі книг. Можливість поповнення балансу (демо-режим).
- **Соціальна взаємодія:** написання відгуків та оцінювання книг; можливість дарувати книги іншим користувачам за нікнеймом; відстеження "стріку" (днів поспіль) відвідування сайту.
- **Профіль:** кастомізація аватара, зміна пароля, перегляд статистики читання.
- **Пошук та фільтри:** зручний пошук за назвою/автором та фільтрація за жанрами і ціною.

### Для адміністраторів
- **Керування контентом:** додавання нових книг (завантаження обкладинки та PDF-файлу).
- **Редагування:** видалення книг з бібліотеки.

### Інтерфейс
- Адаптивний дизайн.
- Темна та світла теми.

## Технологічний стек
- **Backend:** Python 3.9, Flask (Blueprints architecture).
- **Database:** MySQL 8.0.
- **ORM/DB Access:** MySQL Connector (нативні SQL запити).
- **Frontend:** HTML5, CSS3, JavaScript, Jinja2 Templates.
- **Authentication:** Flask-Login.
- **Forms:** Flask-WTF.
- **Containerization:** Docker, Docker Compose.

## Встановлення та запуск

### Передумови
Перед початком переконайтеся, що у вас встановлено:
- [Docker](https://www.docker.com/get-started/)
- [Docker Compose](https://docs.docker.com/compose/install/)

### Крок 1: Клонування репозиторію
```bash
git clone [https://github.com/RussuAn/InkFlow.git]
cd InkFlow
```

### Крок 2: Налаштування змінних оточення
Створіть файл .env у кореневій директорії проєкту та заповніть його налаштуваннями:
```bash
SECRET_KEY=your_secret_key_here
MYSQL_HOST=db
MYSQL_USER=user
MYSQL_PASSWORD=password
MYSQL_DB=inkflow_db
```

### Крок 3: Запуск через Docker
Використовуйте Docker Compose для збірки та запуску контейнерів (додаток + база даних + phpMyAdmin):
```bash
docker-compose up --build
```
Після успішного запуску веб-сервер буде доступний за адресою: http://localhost:5000. А також phpMyAdmin для керування базою даних: http://localhost:8080

## Структура проєкту
```bash
InkFlow/
├── app/
│   ├── core/           # Конфігурація, логер, підключення до БД
│   ├── forms/          # WTF-форми (Login, Register, Book, Review)
│   ├── frontend/       
│   │   ├── static/     # CSS, JS, зображення, завантажені файли
│   │   └── templates/  # HTML шаблони (Jinja2)
│   ├── models/         # Логіка роботи з БД (User, Book, Library, etc.)
│   ├── routes/         # Маршрути Flask (Auth, Books, User, Reader)
│   └── __init__.py     # Ініціалізація додатку
├── logs/               # Логи додатку
├── docker-compose.yml  # Конфігурація Docker сервісів
├── Dockerfile          # Інструкції для збірки образу Flask
├── requirements.txt    # Залежності Python
├── run.py              # Точка входу
└── schema.sql          # Схема бази даних (таблиці)
```

## Адміністрування
За замовчуванням всі нові користувачі мають роль user. Щоб надати права адміністратора:
1. Зареєструйте нового користувача через веб-інтерфейс.
2. Зайдіть у phpMyAdmin.
3. Відкрийте таблицю users.
4. Знайдіть свого користувача і змініть поле role з 'user' на 'admin'.
5. Після цього у випадаючому меню профілю з'явиться кнопка "+ Додати книгу".
